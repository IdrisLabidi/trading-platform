# Plateforme de négociation boursière (Trading Platform)

Ce projet est développé dans le cadre d'un stage d'été au sein de l'entreprise [Network Associates](https://www.networkassociates.tn/), spécialisée dans les solutions IT.

L'objectif principal de ce projet est de créer une plateforme de négociation boursière permettant aux utilisateurs 
d'acheter et de vendre des actions, de suivre les tendances du marché et d'analyser les performances de leurs investissements.

Ce projet est une opportunité pour nous de mettre en pratique des notions avancées d'architecture logicielle 
et de cloud computing, notamment :

- L'implémentation d'une architecture microservices scalable et hautement disponible.
- L'utilisation de Docker pour la conteneurisation des services.
- Le déploiement sur Kubernetes pour la gestion et l'orchestration des conteneurs.
